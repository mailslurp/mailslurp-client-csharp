    List records of expired inboxes

    Declaration
    Task<IGetExpiredInboxesApiResponse> GetExpiredInboxesAsync(Option<int> page = default, Option<int> size = default, Option<string> sort = default, Option<DateTime> since = default, Option<DateTime> before = default, Option<Guid> inboxId = default, CancellationToken cancellationToken = default)
    Parameters
    Type Name Description
    Option<int> page

    Optional page index in inbox sent email list pagination (optional, default to 0)
    Option<int> size

    Optional page size in inbox sent email list pagination (optional, default to 20)
    Option<string> sort

    Optional createdAt sort direction ASC or DESC (optional, default to ASC)
    Option<DateTime> since

    Filter by created at after the given timestamp (optional)
    Option<DateTime> before

    Filter by created at before the given timestamp (optional)
    Option<Guid> inboxId

    Optionally filter by inbox ID (optional)
    CancellationToken cancellationToken

    Cancellation Token to cancel the request.
    Returns
    Type Description
    Task<IGetExpiredInboxesApiResponse>

    Task<IGetExpiredInboxesApiResponse>
    Remarks

    Inboxes created with an expiration date will expire after the given date. An ExpiredInboxRecord is created that records the inboxes old ID and email address. You can still read emails in the inbox (using the inboxes old ID) but the email address associated with the inbox can no longer send or receive emails. Fetch expired inbox records to view the old inboxes properties
